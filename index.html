Theme Park Trekker

Github - Theme Park Trekker

This is a site that was built around traveling to Walt Disney World. I've been a life long fan of Disney and try to travel to it every year! This site offers up frequent posts for tips for traveling to Disney World, as well as a small trip planner! You can search for flights, lodging, and rental vehicles as well! The goal is for you to be able to plan for your next trip in the easiest way possible!

This site is powered by Typescript, postgreSQL, Bcrypt, Cors, Express, Jsonwebtoken, and Nodemon. Please feel free to check out each of the repos that power this application or even visit the site itself!

Giftify

Giftify Github - Giftify

Ever have a hard time find that perfect gift? Check out Gifity! This site is built entirely around building ideas and lists to help you find that perfect give for the hard to shop for person in your life. The best part about this site is that you can see the ideas that other users have come up with so you can add them to your list too! Finding that perfect gift has never been easier!

This multi-contributor MERN stack project is powered by Javascript, Express, Mongoose, Jsonwebtoken, Bcrypt, Sass, React, Axios, and many others. Please take a look through our repo and see what brings this wonderful site to life!

Nibble Or Gorge

Nibble Or Gorge Github - Nibble Or Gorge

This is a recipe website that is allergy friendly. The biggest focus was to allow people whom suffer from food allergies to be able to find new and creative meals to eat! Having experience with how difficult it can be someone to find a reliable and consistent source of creative meals when you have restrictions is very difficult.

This site was built with technologies such as Javascript, NodeJS, Express, Mongoose, Bcrypt, React, and more! Please check out the Github repo above to learn more!
